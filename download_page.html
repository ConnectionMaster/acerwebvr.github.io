01

Install Acer VR Utility for Chrome Browser WebVR (Experimental)

  • Make sure you have installed Google Chrome browser.
  • Go to https://www.acer.com/ac/en/US/content/support-product/7405?b=1
  • Download and install Acer VR Utility for Chrome Browser WebVR (Experimental)
02

You must have the Mixed Reality Portal running

  • Start the Mixed Reality Portal.
03

Launching WebVR

  • Launch Google Chrome browser
  • Navigate to a WebVR enabled page. Or click the Acer VR Utility for Chrome Browser WebVR icon in the Chrome browser upper right corner. Acer WebVR Start Page will show up. You can start with the recommended WebVR experiences listed in the portal.
